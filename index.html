CSS Reference

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

There are three types of CSS which are given below: Inline CSS. Internal or Embedded CSS. External CSS

  • The name cascading comes from the specified priority scheme to determine which style rule applies if more than one rule matches a particular element. This cascading priority scheme is predictable.
  • The CSS specifications are maintained by the World Wide Web Consortium (W3C). Internet media type (MIME type) text/css is registered for use with CSS by RFC 2318 (March 1998). The W3C operates a free CSS validation service for CSS documents.
!important

Plays a role when users include their own stylesheet. If there is a conflict where the style of the page author and the user for the same element does not match, then! Important allows the style to take precedence.

  • ! important is added to the author's style - the author's style will be applied.
  • ! important is added to the custom style - the custom style will be applied.
  • ! important is not present in both the author's style and the user's style - the author's style will be applied.
-moz-border-bottom-colors

Sets the set of colors for the bottom border of an element. If the border is thicker than one pixel, you can set a custom color for each line of pixels.

none - No color, or using the color specified by the border-color property.

color - A color value in any format valid for CSS. You can also use the transparent value to specify transparency.

::after
An element that is used to display the desired text after the content of the element to which it is added. The :: after pseudo-element works in conjunction with the content property. element :: after {content: "text"} When adding :: after to a block element, the display property value can only be: block, inline, none, marker. All other values will be treated as block.
::before

the :: before element is applied to display the desired content before the content of the element to which it is added. Works in conjunction with the content property.

When adding :: before to a block element, the display property value can only be: block, inline, none, marker. All other values will be treated as block.

When adding :: before to an inline element, display is limited to inline and none. All others will be treated as inline.

:first-child
The: first-child pseudo-class applies styling to the first child of its parent.

The :first-child CSS pseudo-class represents the first element among a group of sibling elements., element: first-child {...} In this example, the: first-child pseudo-class is added to the selector

By simply assigning it a value. For example, p:first-child { color: lime; } As originally defined, the selected element had to have a parent. Beginning with Selectors Level 4, this is no longer required.

:target

To go to the selected fragment of the document, the # symbol is written in the address and the name of the identifier is indicated. For example, at http://www.w3.org/TR/css3-selectors/#target-pseudo, you navigate to an element whose id attribute is target-pseudo.

This address record is called the "target item". The: target pseudo-class applies to the target element, in other words, the identifier that is specified in the browser's address bar.

element: target {...}

In this example, the target element is highlighted with a background color.

@import

The @import rule allows you to import the contents of a CSS file into the current style sheet. @import is not allowed to be inserted after any declaration other than @charset or other @import.

Examples: @import url ("filename") [media types]; @import "filename" [media types];

background-position

pecifies the starting position of the background image set using the background-image property.

background-position: [left | center | right | <проценты> | <значение>] || [top | center | bottom | <проценты> | <значение>] | inherit

In CSS3, it is permissible to specify multiple values for each background, listing the values separated by commas.

background-position: <позиция>[, <позиция>]*
border-image

Used to display a hand-drawn border around an element. The border thickness is set by the border property, and if border: 0 is specified, then the border is not displayed. With other values of border, the drawing always takes precedence. The background, if set via the background property, is displayed below the frame.

border-image: none | [ [<число> | <проценты>]{1,4} [/ <толщина>{1,4}]? ] && [stretch | repeat | round]{0,2} <проценты> | <значение>] | inherit
box-sizing
Применяется для изменения алгоритма расчета ширины и высоты элемента. box-sizing: content-box | border-box | padding-box | inherit CONTENT-BOX - Based on CSS standards, the width and height properties set the width and height of the content and do not include padding, margin, and border values. BORDER-BOX - The width and height properties include margin and border values, but not margins. This model is used by the Internet Exporer browser in incompatibility mode. PADDING-BOX - The width and height properties include margin values, but not margins and borders. inherit Inherits the value of the parent.
font-size
Determines the font size of the element. The size can be set in several ways. A set of constants (xx-small, x-small, small, medium, large, x-large, xx-large) sets the size, which is called absolute. list-style
A universal property that allows you to simultaneously set the style of the marker, its position, and the image that will be used as a marker. For details, see the information on each of the list-style-type, list-style-position, and list-style-image properties separately.
